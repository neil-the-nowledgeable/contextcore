Common Scenarios

"What's the status of the checkout feature?"

contextcore task list --project checkout --status in_progress
Instant answer with timestamps, no interrupting developers.

"Why is velocity down this sprint?"

contextcore metrics blocked --project checkout --days 14
See exactly which tasks were blocked and for how long. Data, not guesses.

"When will this be done?"

Dashboard: Project Progress > Burndown Chart
Based on actual velocity, not optimistic estimates.

"What did the team accomplish this quarter?"

contextcore metrics summary --project all --days 90
Aggregate metrics across projects. Perfect for stakeholder reviews.

Quick Start for Teams

# Install ContextCore pip3 install contextcore # Start the observability stack make full-setup # Track your first task contextcore task start --id TASK-001 --title "Setup project tracking" --type task # Mark it in progress contextcore task update --id TASK-001 --status in_progress # Complete it contextcore task complete --id TASK-001 # View metrics contextcore metrics summary --project my-project # Open the dashboard open http://localhost:3000

For Existing Projects

# Generate demo data to see what dashboards look like contextcore demo generate --project my-project # Load to Tempo contextcore demo load --file ./demo_output/demo_spans.json # Provision dashboards to Grafana contextcore dashboards provision # Then start tracking real work
