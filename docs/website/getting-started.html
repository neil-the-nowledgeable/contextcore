Quick Start

Option 1: TUI Wizard (Recommended)

Interactive terminal interface that guides you through setup.

# Launch the wizard contextcore tui launch # Jump to installation contextcore tui install # Check service health contextcore tui status

Option 2: One Command

Full stack deployment with verification.

# Deploy everything make full-setup # Verify installation contextcore install verify

Option 3: Generate Custom Script

Create a tailored installation script for your environment.

# Interactive script generator contextcore tui generate-script

Verify Your Installation

# Full verification with telemetry contextcore install verify # Quick status check contextcore install status # View dashboard open http://localhost:3000/d/contextcore-installation
